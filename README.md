# Simple NFT Minter

This repository provides a high-quality, flat-structured implementation of an ERC-721 NFT contract. It is ideal for developers looking to launch a digital art collection or learn the fundamentals of Non-Fungible Tokens.

## Features
* **ERC-721 Standard:** Fully compatible with NFT marketplaces like OpenSea and Rarible.
* **Auto-Incrementing IDs:** Safely handles Token IDs during the minting process.
* **Metadata Support:** Allows for a Base URI to link your NFTs to IPFS or centralized storage.
* **Ownable:** Secure administrative functions (like withdrawing funds or changing URIs).

## Getting Started
1. Open [Remix IDE](https://remix.ethereum.org/).
2. Create a file named `SimpleNFT.sol`.
3. Compile using Solidity compiler version `0.8.20`.
4. Deploy and call the `safeMint` function to create your first NFT.
